## 📚 What You'll Learn

| Week | Topic | Lessons |
|------|-------|---------|
| 1 | **AI Fundamentals** — What is AI, history, types, models | 1-5 |
| 2 | **LLMs** — Tokens, context window, current models, prompts | 6-10 |
| 3 | **Infrastructure** — Providers, APIs, free options, Ollama | 11-15 |
| 4 | **Agents** — What agents are, components, tools, memory | 16-20 |
| 5 | **OpenClaw Practical** — Installation, config, Telegram bot | 21-25 |
| 6 | **Put It to Work** — Skills, automations, security | 26-30 |

## ⚡ How It Works

This course is **generated by AI and published by AI agents**:

1. Content is created by Claude (Anthropic)
2. An OpenClaw agent (Xani) translates and publishes
3. Hugo builds the static site
4. GitHub Actions deploys automatically

All supervised by a human editor with 20+ years of IT experience.

## 🛠️ Tech Stack

- **Hugo** — Static site generator
- **PaperMod** — Theme
- **GitHub Pages** — Free hosting
- **GitHub Actions** — CI/CD
- **OpenClaw** — Agent framework for publishing
